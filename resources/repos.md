# 开源项目清单

值得学习和参考的 Agent 开源项目。

---

## 框架类

### 1. LangChain
- **链接**：https://github.com/langchain-ai/langchain
- **语言**：Python / TypeScript
- **Star**：~90k+
- **特点**：最流行的 LLM 应用框架，Agent 功能完善
- **推荐阅读**：
  - `langchain/agents/` 目录
  - Agent 的核心实现和设计模式
- **学习状态**：📝 待学习

### 2. LlamaIndex
- **链接**：https://github.com/run-llama/llama_index
- **语言**：Python
- **Star**：~35k+
- **特点**：专注于数据索引和 RAG，也有 Agent 功能
- **推荐阅读**：
  - Agent 实现
  - 记忆系统设计
- **学习状态**：📝 待学习

### 3. AutoGPT
- **链接**：https://github.com/Significant-Gravitas/AutoGPT
- **语言**：Python
- **Star**：~170k+
- **特点**：早期的自主 Agent 实验，影响力巨大
- **推荐阅读**：
  - 任务分解逻辑
  - 记忆管理
- **学习状态**：📝 待学习

### 4. MetaGPT
- **链接**：https://github.com/geekan/MetaGPT
- **语言**：Python
- **Star**：~40k+
- **特点**：多 Agent 协作框架，模拟软件公司
- **推荐阅读**：
  - 多 Agent 协作模式
  - 角色定义
- **学习状态**：📝 待学习

### 5. AutoGen
- **链接**：https://github.com/microsoft/autogen
- **语言**：Python
- **Star**：~30k+
- **特点**：Microsoft 出品，多 Agent 对话框架
- **推荐阅读**：
  - Agent 通信协议
  - 对话管理
- **学习状态**：📝 待学习
