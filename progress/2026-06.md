### 学到的内容
- Agent 的核心架构：感知-决策-执行循环
- 学习 Agent 应该从手写原理开始，而非直接用框架
- ReAct 模式（Reasoning + Acting）是理解 Agent 的关键
- **工具 vs Agent 的本质区别**：工具是被动函数，Agent 是主动决策者
- **OpenAI Function Calling 工作原理**：
  1. 把工具 JSON Schema 传给 LLM
  2. LLM 返回 tool_calls 请求（只是请求，不执行）
  3. 我们的代码执行工具，把结果以 role="tool" 消息返回
  4. LLM 根据结果继续推理，直到输出最终答案
- **messages 列表是 Agent 的记忆**：每次 API 调用都把完整历史传进去，LLM 本身是无状态的
- **AST 白名单安全计算**：比 eval() 更安全，防止代码注入
